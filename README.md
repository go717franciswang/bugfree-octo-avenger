## Development
check style
```sh
checkstyle HelloWorld.java
```

find bugs
```sh
findbugs HelloWorld.class
```

classpath wrappers for Windows
```sh
javac-algs4 HelloWorld.java
java-algs4 HelloWorld
checkstyle-algs4 HelloWorld.java
findbugs-algs4 HelloWorld.class
```
